registry flag

flag:

  • npm login --registry (registry-url)
  • npm publish --registry (registry-url)
  • npm set registry http://localhost:4873

examples:

  • https://registry.npmjs.org/
  • https://registry.reggi.com/ (proxy)
  • http://localhost:4873/ (proxy)
